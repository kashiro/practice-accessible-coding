tabindex=0

Elements which have tabindex=0 attribute can be focused.

li tag can not be focused by default

  • apple
  • banana
  • mango

li tag (with tabindex=0) can be focused.

  • apple
  • banana
  • mango

li tag (with tabindex=1+) can be focused in value order.

  • apple (tabindex="2")
  • banana (tabindex="4")
  • mango (tabindex="3")

li tag (with tabindex=-1) can be focused using javascript.

  • apple
  • banana
  • mango
